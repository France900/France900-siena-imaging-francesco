al fine di far funzionare questa repo, una volta clonata sul pc tramite github desktop è sufficiente fare doppio click sul file index.html

nell'eventualità che non basti sarà necessario: 
intallare node.js https://nodejs.org/it/download/

proseguendo con i comandi per avviare la repo. questi sono gli stessi comandi necessari per modificare la repo

nel terminale allinterno della cartella della repo:
localmente installare laravel mix (https://laravel-mix.com/docs/6.0/installation) con i seguenti passi:
1)  npm init -y
2)  npm install laravel-mix --save-dev
3)  npx mix watch
